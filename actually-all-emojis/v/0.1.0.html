Actually All Emojis

An up-to-date package for espanso which provides triggers for all emojis (
v14.0
). Emojis and codes fetched from Unicode.org

Usage

To use this package, just type the name of the emoji between two colons. For example, to use the
smile
emoji , you can type:
:smile:
A full list of emojis and their CLDR Short Names can be found here
Note: There are two options for this package. One with spaces between words, and one with underscores. This version uses underscores For example
:kissing_face:
To install the version with spaces use
actually-all-emojis-spaces
