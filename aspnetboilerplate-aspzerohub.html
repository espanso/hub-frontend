aspnetboilerplate-aspzerohub
By Can Maviolu

A package to include aspnetboilerplate and asp zero shortcut code generation in espanso.

code
aspnetboilerplate
aspzero
aspnetboilerplate-espanso
aspzerohub-espanso
aspnetboilerplate-aspzerohub-espanso

Paste this command in a terminal to install the package

espanso install aspnetboilerplate-aspzerohub
