HTML Named Entities

html-entities-unicode
By enky, package built by Stephen Meech

A character entity decoder, converting HTML/XML named character entities into their corresponding Unicode characters.

html
unicode
symbols
characters

Paste this command in a terminal to install the package

espanso install html-entities-unicode

Source

HTML entities unicode

Coincidentally, written at the same time as the similar named-html-entities, but supplies characters written via Espanso Unicode escape format.
Uses compact inline Flow style.
